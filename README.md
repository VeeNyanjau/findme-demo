FindMe is a personal safety application designed to help users share their location and status in emergencies. It uses Firebase for real-time data synchronization.

‚ú® Features
Real-time Location Sharing: Updates location to Firebase Realtime Database.
Offline Mode: Sends SMS alerts with location coordinates when internet is unavailable.
Emergency Contacts: Manage standard and emergency contacts.
Background Service: Monitors status even when the app is minimized.
Secure Auth: Firebase Authentication for user management.

Prerequisites
Android Studio (Koala or newer recommended)

Installation

Open in Android Studio

Launch Android Studio

Click Open an existing project and select the cloned folder

Build the project

Connect your Android device via USB or start an Android emulator

Click the Run button (Shift + F10) in Android Studio

Wait for the app to install and launch on your device/emulator

Set up the app

Open the app on your device

Create a new community or join an existing one

Follow the on-screen instructions to start sharing your location and connecting with members
